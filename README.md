# Jaeun Shin — Personal Academic Website

Hello! This public repository contains the source code for my personal academic website:

https://jaeunshin.github.io

The site is built using the **al-folio** academic website theme:  
https://github.com/alshedivat/al-folio

## About

I am currently an engineering student with a developing interest in research. This website serves as a central place to document my academic work and development over time.

As I progress, I plan to use the site to share:

- research projects
- publications or preprints
- technical notes and posts
- academic activities
- contact information

The site will gradually evolve as my research interests and work develop.

## Deployment

The website is automatically built and deployed using GitHub Pages. Updates to the repository trigger a new build and publication.

## Acknowledgement

The design and structure of the site are based on the open-source **al-folio** theme.
